# Installation

This page documents how to install **eb-optimization** for different use cases, including
end users, evaluators, and contributors working inside the Electric Barometer ecosystem.

Install eb-optimization into the environment where you will **construct or consume policy
artifacts** (e.g., CostRatioPolicy, TauPolicy, RALPolicy) and run **offline tuning** routines.
For evaluation-time application, eb-optimization is typically used alongside `eb-evaluation`.
